# Security Policy

## Supported versions

Souther is pre-1.0 and still evolving. Only the latest published version receives
security fixes.

| Version | Supported |
| ------- | --------- |
| latest  | ✅        |
| older   | ❌        |

## Reporting a vulnerability

Please report a suspected vulnerability privately through GitHub's
[private vulnerability reporting](https://github.com/kawasima/souther/security/advisories/new)
(the repository's **Security** tab → **Report a vulnerability**). Do not open a public
issue for a suspected vulnerability.

We aim to acknowledge a report within a few days and will coordinate a fix and a
disclosure with you.
